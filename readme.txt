== Description ==
RJ-Contact is a versatile, free WordPress plugin empowering users to effortlessly integrate a customizable contact form into their websites. With an intuitive installation process, this plugin offers a watermark-free experience, allowing users to seamlessly manage and customize form styles. == Installation ==
1. Install the plugin for WordPress plugin directory easily.
2. Deactivate or reactivate the plugin through the 'Plugins' menu in WordPress.

== Features ==
- Easy installation and activation.
- Create and manage customizable contact forms.
- Style your forms with custom CSS using the RJ-Style settings.
- No watermarks or restrictions.
- Enhance user interaction on your website effortlessly.

== Frequently Asked Questions ==
= How do I add a contact form to my page or post?
To add a contact form, simply use the shortcode `[ravi_contact_form]` in the content area of your page or post.

= Can I customize the appearance of the contact form?
Yes, you can customize the form's appearance using the RJ-Style settings. Adjust the CSS code to suit your design preferences.

= Is the plugin free to use?
Yes, RJ-Contact is completely free to use, and there are no watermarks or restrictions.
